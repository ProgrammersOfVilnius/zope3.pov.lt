Zope 3.4 KGS buildbot has been shut down

Nobody uses Zope 3.4 any more.

If you do, please upgrade to Bluebream (which is a rebranded direct continuation of the Zope 3 idea), or use the Zope Toolkit (which is a direct continuation of the inner core of Zope 3), or switch to a different framework like Pyramid (which was inspired by a lot of Zope 3 technology, but doesn't have the sharp edges and actually has good documentation).
